## 4. مخطط المستوى الأول للعملية 2.0 و 3.0 إدارة الاكتتاب والوثائق

```mermaid
flowchart LR
    C[العميل]

    D1[(D1<br/>بيانات العملاء)]
    D2[(D2<br/>المنتجات وقواعد التسعير)]
    D3[(D3<br/>الوثائق التأمينية)]
    D6[(D6<br/>الإشعارات وسجل العمليات)]

    P21((2.1<br/>استقبال طلب التأمين))
    P22((2.2<br/>احتساب السعر المبدئي))
    P23((2.3<br/>مراجعة الاكتتاب))
    P31((3.1<br/>إصدار الوثيقة))
    P32((3.2<br/>تجديد أو تعديل أو إلغاء الوثيقة))

    C -- بيانات الطلب ومرفقاته --> P21
    P21 -- حفظ بيانات العميل المقدم --> D1
    P21 -- قراءة منتج التأمين --> D2
    D2 -- قواعد التسعير وحدود التغطية --> P22
    P22 -- العرض السعري المبدئي --> U
    U -- قرار الاكتتاب --> P23
    P23 -- طلب معتمد --> P31
    P23 -- نتيجة الاكتتاب للعميل --> C
    P31 -- حفظ بيانات الوثيقة --> D3
    P31 -- إشعار إصدار الوثيقة --> C
    U -- طلب تجديد أو تعديل أو إلغاء --> P32
    D3 -- بيانات الوثيقة الحالية --> P32
    P32 -- تحديث سجل الوثيقة --> D3
    P21 -- تسجيل استقبال الطلب --> D6
    P22 -- تسجيل عملية التسعير --> D6
    P23 -- تسجيل قرار الاكتتاب --> D6
    P31 -- تسجيل إصدار الوثيقة --> D6
    P32 -- تسجيل عملية تعديل الوثيقة --> D6
```
