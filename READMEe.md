# مخططات تدفق البيانات للفصل الثالث

يحتوي هذا الملف على مخططات تدفق البيانات المقترحة بصيغة `Mermaid`، ويمكن نسخ كل مخطط وتحويله إلى صورة لإدراجه في وثيقة المشروع.

## 1. المخطط البيئي

```mermaid
flowchart LR
    C[العميل]
    E[موظف التأمين]
    M[الإدارة]
    X[جهات خارجية<br/>بوابة دفع - خبير معاينة - جهات رقابية]
    S((0<br/>منصة أمان لإدارة التأمينات))

    C -- طلب تسجيل / طلب تأمين / رفع وثائق / تسجيل مطالبة / سداد قسط / اعتراض --> S
    S -- بيانات الوثيقة / حالة المطالبة / إشعارات / إيصال سداد --> C

    E -- إدخال بيانات / مراجعة طلبات / اعتماد وثائق / معالجة مطالبات --> S
    S -- قوائم العمل / بيانات العملاء / نتائج المراجعة / سجل المتابعة --> E

    M -- سياسات / صلاحيات / طلب تقارير / قرارات اعتماد --> S
    S -- تقارير / مؤشرات أداء / تنبيهات / سجل عمليات --> M

    X -- نتائج دفع / تقارير معاينة / بيانات داعمة --> S
    S -- أوامر دفع / طلبات تحقق / ملفات معاينة / تقارير دورية --> X
```

## 3. مخطط المستوى الأول للعملية 1.0 إدارة العملاء والمستخدمين

```mermaid
flowchart LR
    C[العميل]
    E[الموظف]
    D1[(D1<br/>بيانات العملاء والحسابات)]
    D6[(D6<br/>سجل العمليات)]

    P11((1.1<br/>إنشاء حساب))
    P12((1.2<br/>التحقق من البيانات))
    P13((1.3<br/>تحديث الملف الشخصي))
    P14((1.4<br/>إدارة الأدوار والصلاحيات))

    C --> P11 --> P12 --> D1
    E --> P11
    E --> P13
    E --> P14
    D1 --> P13 --> D1
    P11 --> D6
    P12 --> D6
    P13 --> D6
    P14 --> D6
    P12 --> C
    P13 --> C
```

## 4. مخطط المستوى الأول للعملية 2.0 و 3.0 إدارة الاكتتاب والوثائق

```mermaid
flowchart LR
    C[العميل]
    U[موظف الاكتتاب]
    D1[(D1<br/>بيانات العملاء)]
    D2[(D2<br/>المنتجات وقواعد التسعير)]
    D3[(D3<br/>الوثائق التأمينية)]
    D6[(D6<br/>سجل العمليات)]

    P21((2.1<br/>استقبال طلب التأمين))
    P22((2.2<br/>احتساب السعر المبدئي))
    P23((2.3<br/>مراجعة الاكتتاب))
    P31((3.1<br/>إصدار الوثيقة))
    P32((3.2<br/>تجديد أو تعديل أو إلغاء الوثيقة))

    C --> P21
    P21 --> D1
    P21 --> D2
    D2 --> P22
    P22 --> U
    U --> P23
    P23 --> P31
    P23 --> C
    P31 --> D3
    P31 --> C
    U --> P32
    D3 --> P32 --> D3
    P21 --> D6
    P22 --> D6
    P23 --> D6
    P31 --> D6
    P32 --> D6
```
